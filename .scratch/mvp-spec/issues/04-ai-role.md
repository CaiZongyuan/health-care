# 04 — AI 角色：医生辅助生成随访草稿

Type: grilling
Status: resolved
Blocked by: none

## Question

Vercel AI SDK 在 MVP 里到底用在哪里？医疗风险怎么控制？

## Answer

**医生辅助：AI 生成随访草稿。** 在医生工作台，AI 把某患者近期血压趋势 + 症状 + 用药依从性汇总成一段"随访要点/草稿"，医生**审核修改后**再发给患者（置顶患者首页）。

- **医疗安全约束**（写入 map Notes）：AI 输出始终有人（医生）在回路把关；患者侧不直接收到 AI 的"医疗建议"，只看到数据与医生确认后的反馈。
- 不做：患者侧 AI 健康问答、患者侧异常"医疗建议"——留待有免责与合规保障后再议。
- 解锁：AI 提供方/模型选型（07）、AI SDK 运行时可行性（06）；工作台需要"生成草稿"入口与编辑流转。
