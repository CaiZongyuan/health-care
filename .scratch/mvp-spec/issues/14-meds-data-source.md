# 14 — 用药待办的数据来源

Type: grilling
Status: open
Blocked by: none

## Question

首页"今日用药"待办的数据从哪来？

- 候选 A：患者（或后续家属）在"我的"里**手动维护**用药清单（名称/剂量/时段）→ 系统每天生成待办。
- 候选 B：医生在工作台**开方/维护**患者用药 → 同步成待办。
- MVP 倾向：**患者手填为主**（最简、不依赖医生录入动力），医生可在反馈里口头提醒；医生结构化开方留待 B2B 阶段。

决策后解锁：用药数据模型（药品/剂量/时段/是否服用）、与"我的-长期用药"是否合并、提醒机制（见 map fog）。
