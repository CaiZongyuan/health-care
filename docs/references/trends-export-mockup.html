二、图表候选（解决"看不懂"，多几种）

A增强折线图（日期轴 + 正常带 + 标注）

在当前折线基础上：横轴标日期、纵轴标数值、正常范围绿带、最高/最低点标注。最直观，推荐保留。

B散点分布图（每次一个点）

每次测量一个点（按日期排开），颜色表示达标/偏高。适合看"波动幅度"和"哪几次超标"。

达标正常高值偏高

C每日均值柱状图（近 14 天）

每天一个均值柱（高压/低压），上方有达标参考线。适合看"每天整体水平"的趋势。

D达标率 + 区间分布

环形图显示达标率；下方柱状显示各区间占比（偏低/正常/正常高值/偏高）。一眼看"整体控制得怎么样"。

E日历热力图（近 30 天）

类似 GitHub 贡献图：每天一格，颜色=当日达标情况（绿=达标、黄=正常高值、红=偏高、灰=无数据）。一眼看"哪几天不好"。

达标正常高值偏高无数据
三、导出给医生（优雅、便于分析）

G医生报告（A4 / 可打印成 PDF）

一页 A4 排版：患者档案 + 关键统计 + 趋势图 + 异常清单 + 用药 + 小结。点"打印/导出PDF"用浏览器另存 PDF。医生友好。
